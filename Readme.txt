왼쪽 마우스 버튼을 누르면: 
 나무 몸통이 매 프레임마다 자란다
다시 왼쪽 마우스 버튼을 누르면:
 나무를 생성한다.

왼쪽/오른쪽 화살표 키: 나무 선택
아래쪽 화살표 키: 감상 모드 ON(선택한 나무가 갈색으로 표시되지 않음)
위쪽 화살표 키: 감상 모드 OFF(선택한 나무가 갈색으로 표시됨)

q/Q: 종료
b/B: 선택한 나무에 꽃 피우기/제거
a/A: 모든 나무에 꽃 피우기
d/D: 모든 나무의 꽃 제거
s/S: 화면 초기화(모든 나무 지우기)
r/R: 랜덤 나무 그리기
e/E: 선택한 나무 지우기
